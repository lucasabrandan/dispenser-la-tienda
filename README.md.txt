# Dispenser La Tienda

Sistema profesional para gestionar clientes, sedes, equipos (dispensers), servicios técnicos, repuestos, stock y recordatorios de mantenimiento.

## Stack
- Java 21
- Spring Boot
- Maven
- PostgreSQL (próxima etapa)

## Cómo ejecutar
1. Abrir el proyecto en IntelliJ IDEA
2. Ejecutar `DispenserLaTiendaApplication`
3. Probar: `GET http://localhost:8080/health`
